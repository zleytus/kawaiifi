# Resource Data

`manuf` is generated from Wireshark/TShark with:

```sh
tshark -G manuf > manuf
```

The generated `manuf` data is derived from Wireshark's compiled-in
manufacturer data. Wireshark is licensed under GPL-2.0-or-later, and KawaiiFi
is licensed under GPL-3.0-or-later.

See <https://www.wireshark.org/> and
<https://gitlab.com/wireshark/wireshark>.
